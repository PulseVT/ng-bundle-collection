Static_methods

Description

Static properties of ng-bundle-collection.Collection

Methods

  • cancelAllRequests()

    Cancels all pending requests in all instances of ng-bundle-collection.Collection by calling cancelAllRequests methods

  • clear(settings)

    Clears all instances (clears arr and objById) of ng-bundle-collection.Collection by calling clear methods

    Parameters
    ParamTypeDetails
    settingsobject

    Settings of clearing (object with boolean flags):

    • withExtendFns=false - whether to remove all configured extending functions (collection.extendFns)
    • withInterceptors=false - whether to remove all collection.interceptors

  • invalidate()

    Invalidates all instances (clears cache) of ng-bundle-collection.Collection by calling invalidate methods

Properties

  • instances

    Array of all ng-bundle-collection.Collection instances
